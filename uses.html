Software I use, gadgets I love, and other things I recommend.

I get asked a lot about the things I use to build software, stay productive, or buy to fool myself into thinking I'm being productive when I'm really just procrastinating. Here's a big list of all of my favorite stuff.

Workstation

2018 MacBook Air 13-inch

Lightweight and portable with a Dual-Core Intel Core i5 and Retina Display, great for on-the-go work and general productivity.

2015 MacBook Pro 15-inch

A robust workhorse with a Quad-Core Intel Core i7 and AMD Radeon GPU, ideal for demanding tasks like coding, design, and video editing.

Dell Vostro 15-inch

A solid choice for general use, featuring a 12th gen Intel Core i5 and integrated graphics, perfect for development and multitasking.

Dell Optiplex 390

A reliable desktop with a 2nd gen Intel Core i5 and AMD Radeon GPU, great for basic tasks and light graphic work.

Development tools

Alfred

A productivity app for macOS that boosts efficiency with custom workflows and quick search capabilities.

VSCode

A powerful, lightweight code editor with a wide range of extensions and customization options.

Postman

An API testing and collaboration tool that simplifies building, testing, and documenting APIs.

Warp

A modern terminal built for developers, offering faster workflows and advanced customization.

Zsh

A customizable shell that enhances the terminal experience with features like auto-suggestions and powerful scripting.

Microsoft Edge

A RAM-efficient browser optimized for development with powerful tools like WebView2 and seamless integration with Microsoft services.

One Dark Modern

A sleek, modern VS Code theme that enhances readability and focus during long coding sessions.

Design

Canva

An easy-to-use design tool for creating stunning graphics quickly without needing advanced design skills.

Photoshop

The industry standard for image editing, perfect for creating web graphics and high-quality photo manipulation.

Figma

A collaborative design tool for creating user interfaces, prototypes, and wireframes efficiently in teams.

Davinci Resolve

A professional-grade video editing software that excels at color correction and motion graphics.

Blender

An open-source 3D modeling software ideal for creating detailed 3D assets and animations.

Iclone 8

A robust tool for creating realistic animations and cinematic sequences.

Character Creator 4

A comprehensive tool for crafting lifelike 3D characters with incredible detail.

Productivity

Typora

A minimalist markdown editor that allows you to write and format markdown files with ease.

Obsidian

A powerful knowledge management app that helps you store, organize, and link ideas.
